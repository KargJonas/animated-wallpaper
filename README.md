# animated-wallpaper
#### This is a simple script to run gif wallpapers using `feh`.

### Dependencies:
- feh
- graphicsmagick

### To prepare a gif for use with this script, use the `convert.sh` script:
```bash
./convert.sh my_awesome_wallpaper.gif output_folder
```

### To run a wallpaper:
```bash
./wallpaper.sh output_folder
```
